Interactive Object Counting

Eagle is an object counting that can first detect then count any objects through visual prompting, which is highlighted by the following features:

  • Open-Set: Eagle possess the capacity to count any object, without constraints on predefined categories.
  • Visual Promptable: Users can provide visual examples to specify the objects for counting.
  • Untuitive Visual Feedback: Eagle is a detection-based model that allows for visual feedback (i.e. detected boxes), enabling users to assess the accuracy of the result.
  • Interactive: Users can actively participate in the counting process to rectify any errors.

Application Forms

Eagle is designed for counting, meanwhile, since it is an open-set detection model, it can be used for any detection task, as well as for annotation scenarios.

Object Counting

Eagle is a fundation model for object counting that count anything including but not limited to Agriculture, Industry, Livestock, Biology, Medical, Retail, Electronic, Transportation, Logistics, Human, etc.

Automatic Annotation

Eagle can also serve as an open-set object detector. It process exponential zero-shot detection capability, and offers strong performance in dense and overlapping scenes.

How to use the Demo

If you are new to Eagle, we recommend you to watch the demo video first, so you can know Eagle better.

Demo

  • Step1: Upload target image and reference image to Input Detection Image and Input Reference Image, respectively. You are free to use two different images and Eagle will work in workflow3 (Checkout our Technical Report for more details).
  • Step2: Draw box or point on reference image. To draw box, you need to click on the upper left point of the box, and then the lower right point to draw a box. To draw point, first select the Draw Point option and then draw point.
  • Step3: Run Eagle.
  • Step4: If the detection results is not satisfying, e.g. there are missing or mis-detected objects, you can click Draw Negative to add negative prompt, so that Eagle will work in workflow2.
  • If the guidence is still confusing, we recommend watching the demo video.
